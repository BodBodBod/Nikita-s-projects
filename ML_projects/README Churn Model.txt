Обучена модель оттока сотрудников на данных ученых IBM, это решение было частью решения HR-Tech кейса от компании РЖД на финале кейс-чемпионата Changellenge >> Cup Moscow 2023.

В ноутбуке протестированы различные модели: Logistic Regression, Random Forest, SVM, k-NN, Decision Tree Classifier, Gaussian NB. 
На вход модели подаются данные о сотруднике, по которым происходит прогнозирование ухода данного сотрудника.

За основу для этой работы был взят Churn Model пайплайн в открытом доступе.
